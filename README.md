# Todo List Uygulaması

Bu proje, modern web teknolojileri (HTML, CSS, JavaScript) kullanılarak geliştirilmiş, kullanıcıların günlük görevlerini yönetebileceği şık ve işlevsel bir yapılacaklar listesi uygulamasıdır.

## Özellikler

* **Görev Yönetimi:** Kullanıcılar kolayca görev ekleyebilir, tamamlandığında üzerini çizebilir ve istemediği görevleri silebilir.
* **Dinamik Tema Desteği:** Aydınlık (Light) ve Karanlık (Dark) mod arasında geçiş yapabilme özelliği.
* **Modern Tasarım:** CSS değişkenleri kullanılarak hazırlanmış, responsive ve kullanıcı dostu bir arayüz.
* **DOM Manipülasyonu:** JavaScript ile gerçek zamanlı element oluşturma ve yönetim.

##  Kullanılan Teknolojiler

* **HTML5:** Sayfa yapısı ve semantik etiketler.
* **CSS3:** Stil düzenlemeleri, Flexbox, CSS Değişkenleri ve duyarlı tasarım.
* **JavaScript (ES6):** Olay dinleyicileri (event listeners), DOM manipülasyonu ve mantıksal işlemler.
